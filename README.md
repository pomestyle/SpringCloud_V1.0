## ssr-crawler
自动拉取ssr账号并且自动配置，无需手动添加本地文件

无ssr客户端下载的请 [点击]()
---


- 前言

默认情况下我们需要手动在ssr客户端设置账号



每次手动输入账号等操作，很是麻烦。



- 故此




###  ssr 默认解压到c盘根目录即可
   如： c:/vpnSSR
   
   
### 间隔24小时跑一次（每次默认设置60个账号）
  每次若爬取失败连接超时会在1分钟之后重新进行爬取，知道爬取且自动配置成功会停止。
  
  
  
  缺点：没有验证，账号功能，推荐将ssr设置负载均衡模式。
